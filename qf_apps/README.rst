
QuickFeather Applications
=========================

This directory contains applications for specific usage scenarios, which can serve as a useful starting point to develop further applications.

We have the following applications currently available:

- qf_helloworldsw
- qf_helloworldhw
- qf_advancedfpga
- qf_mqttsn_ai_app

The following applications cover the bootloader and its usage:

- qf_bootloader
- qf_bootloader_uart
- qf_loadflash
- qf_loadflash_uart
